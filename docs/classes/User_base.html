select_by()

select_by(string $field) : void

Select all user fields from a property indicated in $field (default 'ID')

The field indicated must be a unique value (ID, user_name, email) and corresponding object property should be setted.

Throws an exeption on failure.

Parameters

string $field

the field name

Throws

\Custom_exception(ID_NON_EXISTENT)

if the ID does not exists

\Custom_exception(USER_NAME_NON_EXISTENT)

if the user name does not exists

\Custom_exception(EMAIL_NON_EXISTENT)

if the email address dows not exists

\Custom_exception(INVALID_PARAMETER)

_insert_on_duplicate()

_insert_on_duplicate(string $table, array $data) : void

Perform an insert query with 'on duplicate key update' clause.

The first parameter contains a tring with the table name, the second one an associative array with th values.

Ex. $table = 'users'; $data = array(

'id' => 1,
'name' => 'test',

); $this->_insert_on_duplicate($table, $data);

produces the following query:

INSERT INTO users (id, name) VALUES ('1', 'test') ON DUPLICATE KEY UPDATE id='1', name='test'

Parameters

string $table

is the table name
array $data

the associative array to insert
