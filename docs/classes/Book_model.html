_insert_on_duplicate()

_insert_on_duplicate( $table,  $data) : void

Perform an insert query with 'on duplicate key update' clause.

The first parameter contains a tring with the table name, the second one an associative array with th values.

Ex. $table = 'users'; $data = array(

'id' => 1,
'name' => 'test',

); $this->_insert_on_duplicate($table, $data);

produces the following query:

INSERT INTO users (id, name) VALUES ('1', 'test') ON DUPLICATE KEY UPDATE id='1', name='test'

Parameters

$table
$data
