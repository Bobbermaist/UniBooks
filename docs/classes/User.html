_try()

_try(string $property_name, string $method_name) : void

Executes a try - catch command through a callback.

A model that may throws exception should be executed with this method.

The first parameter indicates the model name to call (e.g. 'User_model'), the second one the method (like any method or User_model).

It is also possible to pass an arbitrary numer of parameters in addition to these two $this->_try('User_model', 'method_name', $param1, $param2, ...).

In this case all parameters will be passed to the called method.

E.g.

 $this->_try('User_model', 'reset_password', $confirm_code, $new_password);

Corresponds to:

 try {
   $this->User_model->reset_password($confirm_code, $new_password);
 } catch (Custom_exception $e) {
   $this->exception_code = $e->getCode();
   $this->exception_message = $e->getMessage();
 }

The model in this example should be loaded before calling this method, otherwise an INVALID_PARAMETER exception will be thrown.

If the called method raises a Custom_exception, its code will be stored in exception_code property and its message in exception_message.

Parameters

string $property_name

controller's property name
string $method_name

method name

_set_message()

_set_message(string|false $name) : void

Loads a message from message_lang and enqueue this message through "_set_view()".

If an exception was previously catched enqueue the exception message instead the requested message. After sets the exception_message to NO_EXCEPTIONS.

If $name parameter is FALSE (default) won't load any message in case of the is no exception

Parameters

string|false $name

the name of message lang

Throws

\Custom_exception(INVALID_PARAMETER)

if can't loads the requested lang line
